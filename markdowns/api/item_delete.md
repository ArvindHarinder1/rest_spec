# Delete

Delete item.
### Prerequisites
The following **scopes** are required to execute this API: ### HTTP request

```http
DELETE /drive/root
DELETE /drive/items/<id>

```

|:---------------|:--------|:----------|

### Response
If successful, this method returns `204, No Content` response code. It does not return anything in the response body.
