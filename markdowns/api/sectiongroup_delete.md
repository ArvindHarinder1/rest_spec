# Delete

```http
DELETE /users/<objectId>/notes/sectionGroups/<id>
DELETE /drive/root/createdByUser/notes/sectionGroups/<id>
DELETE /drive/root/lastModifiedByUser/notes/sectionGroups/<id>

```

| Name       | Type | Description|
