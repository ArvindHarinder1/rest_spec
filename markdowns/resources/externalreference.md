# ExternalReference resource type

Here is a JSON representation of the resource

```json
{
  "alias": "String",
  "modifiedBy": "String",
  "modifiedDate": "String (timestamp)",
  "previewPriority": "String",
  "type": "String"
}

```

|alias|String||
|modifiedBy|String||

|type|String||
